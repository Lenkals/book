🌐 **[Open the Reference →](https://your-username.github.io/big-ideas-for-young-minds/)**

## ✨ What Is This?

A single-page interactive web app that organizes humanity's most important ideas into a browsable, searchable reference. Every idea includes:

- **Simple Explanation** — written for a 12-year-old
- **Key Concept** — the core idea in one line
- **Real-World Example** — a relatable analogy or scenario
- **Children's Story Ideas** — suggestions for adapting ideas into narratives

## 🚀 Features

- **Instant search** across all 249 ideas — find any thinker, concept, or keyword
- **Collapsible sections** — explore at your own pace
- **Click-to-expand** real-world examples
- **Top 10 ideas** highlighted per subject
- **Story adaptation suggestions** for each subject
- **Fully offline** — single HTML file, no dependencies, no server needed
- **Dark theme** with responsive mobile layout

## 🎯 Who Is This For?

- **Parents & teachers** building educational content for children
- **Students** wanting a quick reference to big ideas across many fields
- **Writers** creating children's books inspired by these ideas
- **Curious minds** who want a broad overview of human knowledge

## 🛠️ Usage

### Option 1: GitHub Pages
1. Fork this repo
2. Enable GitHub Pages in Settings → Pages → Source: `main` branch
3. Visit `https://your-username.github.io/big-ideas-for-young-minds/`

### Option 2: Local
1. Download `index.html`
2. Open it in any browser
3. That's it — no build step, no dependencies

## 🤝 Contributing

Contributions are welcome! You can:

- **Add new ideas** to existing subjects
- **Add new subjects** (e.g., Astronomy, Ecology, Computer Science)
